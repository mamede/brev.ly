## Frontend React

### Funcionalidades e Regras

- [ ]  Deve ser possível criar um link
    - [ ]  Não deve ser possível criar um link com encurtamento mal formatado
    - [ ]  Não deve ser possível criar um link com encurtamento já existente
- [ ]  Deve ser possível deletar um link
- [ ]  Deve ser possível obter a URL original por meio do encurtamento
- [ ]  Deve ser possível listar todas as URL's cadastradas
- [ ]  Deve ser possível incrementar a quantidade de acessos de um link
- [ ]  Deve ser possível baixar um CSV com o relatório dos links criados

### Regras Específicas do Front-end

- [ ]  Aplicação React SPA com Vite
- [ ]  Seguir o layout do Figma
- [ ]  Boa experiência do usuário (empty state, loading, bloqueio de ações)
- [ ]  Foco na responsividade (desktop e mobile)

### Páginas

- [ ]  Página raiz (`/`) - Formulário de cadastro e listagem
- [ ]  Página de redirecionamento (`/:url-encurtada`) - Busca e redireciona
- [ ]  Página 404 - Recurso não encontrado

### Ferramentas Obrigatórias

- [ ]  TypeScript
- [ ]  React
- [ ]  Vite (sem framework)

### Ferramentas Flexíveis (Recomendadas)

- [ ]  TailwindCSS
- [ ]  React Query
- [ ]  React Hook Form
- [ ]  Zod
