## Frontend React

### Funcionalidades e Regras

- [x]  Deve ser possível criar um link
    - [x]  Não deve ser possível criar um link com encurtamento mal formatado
    - [x]  Não deve ser possível criar um link com encurtamento já existente
- [x]  Deve ser possível deletar um link
- [x]  Deve ser possível obter a URL original por meio do encurtamento
- [x]  Deve ser possível listar todas as URL's cadastradas
- [x]  Deve ser possível incrementar a quantidade de acessos de um link
- [x]  Deve ser possível baixar um CSV com o relatório dos links criados

### Regras Específicas do Front-end

- [x]  Aplicação React SPA com Vite
- [x]  Seguir o layout do Figma
- [x]  Boa experiência do usuário (empty state, loading, bloqueio de ações)
- [x]  Foco na responsividade (desktop e mobile)

### Páginas

- [x]  Página raiz (`/`) - Formulário de cadastro e listagem
- [x]  Página de redirecionamento (`/:url-encurtada`) - Busca e redireciona
- [x]  Página 404 - Recurso não encontrado

### Ferramentas Obrigatórias

- [x]  TypeScript
- [x]  React
- [x]  Vite (sem framework)

### Ferramentas Flexíveis (Recomendadas)

- [x]  TailwindCSS
- [x]  React Query
- [x]  React Hook Form
- [x]  Zod
